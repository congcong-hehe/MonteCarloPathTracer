## Introduction
A CPU pathtracer only for triangle mesh.

## Dependencies
* stb_image
* rgbe

## Feature
* vec math
* load obj, mtl
* Monte Carlo Path Tracing
* BVH
* SSE accelerate
* OpenMP
* hdr environment map sample
* importance sample(blinn-sample)
* texture
* hdr important sample
